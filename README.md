### 유튜버 맛집지도
유튜버들이 다녀간 맛집을 보기편하게 지도에 첨부하여 보여주는 서비스입니다.
추가한 유튜버로는 풍자, 공혁준 이 있습니다.
관리자페이지에서 데이터 첨삭을 가능하도록 하였습니다.

### 사용기술
- react 18.2.0, typescript, scss
- axios, redux-toolkit
- aws-ec2
- firebase
